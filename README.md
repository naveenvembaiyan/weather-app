A simple weather app built with React and Vite. Search for a city to view:

- Current temperature and weather condition
- Feels-like temperature, humidity, and wind
- 5-day forecast
- Lightweight UI animations for weather cards and background glow

Data is fetched from the Open-Meteo geocoding and forecast APIs.

## Run locally

```bash
npm install
npm run dev
```

Then open the local URL shown by Vite.
